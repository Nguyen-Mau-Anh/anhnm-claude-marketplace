# /greet

A simple greeting command that demonstrates slash command structure.

## Usage

```
/greet [name]
```

## Description

This command greets the user by name. If no name is provided, it uses a default greeting.

## Instructions

When the user invokes this command:

1. If a name argument is provided, greet them personally: "Hello, [name]! Welcome to Claude Code."
2. If no name is provided, use a friendly default: "Hello! Welcome to Claude Code. How can I help you today?"
3. Offer to help with their current task.

## Examples

- `/greet` - Basic greeting
- `/greet John` - Personal greeting for John
